Radio
User Story #12: Inside the form element, I can select an option from a dropdown that has a corresponding id="dropdown". User Story #13: Inside the form element, I can select a field from one or more groups of radio buttons. Each group should be grouped using the name attribute. User Story #14: Inside the form element, I can select several fields from a series of checkboxes, each of which must have a value attribute. User Story #15: Inside the form element, I am presented with a textarea at the end for additional comments. User Story #16: Inside the form element, I am presented with a button with id="submit" to submit all my inputs.
